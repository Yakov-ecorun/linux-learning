# Linux Permissions Practice

## 📌 Description

Hands-on practice for Linux file permissions, ownership, and users.

## 🔧 What was done

* Created users (user1, user2)
* Created directory /tmp/homework
* Set permissions (400, 600, 700, 755, 777)
* Tested access between users
* Used chmod, chown, chattr

## 📂 Commands used

```bash
chmod 755 /tmp/homework
chown user2 file
chattr +i file
```

## 🎯 Result

User2 can:

* read files
* write to files
* execute scripts
* access directory

## 🚀 Notes

This practice helped understand Linux permissions deeply.
